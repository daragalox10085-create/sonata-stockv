## 🔴 发现的Bug列表

### Bug #1: 导出长图功能失效
**严重程度**: Medium  
**状态**: ❌ 未修复  
**描述**: 点击"导出长图"按钮无任何反应  
**复现步骤**:
1. 打开任意股票详情页
2. 点击顶部"导出长图"按钮
3. 观察：无反应，无下载，无提示

**可能原因**:
- html2canvas库未正确加载
- 导出逻辑未实现
- 缺少错误处理

**修复建议**:
```typescript
// 检查导出功能实现
const exportLongImage = async () => {
  try {
    const element = document.querySelector('.analysis-container');
    if (!element) {
      toast.error('未找到导出内容');
      return;
    }
    
    const canvas = await html2canvas(element, {
      scale: 2,
      useCORS: true,
      logging: false
    });
    
    const link = document.createElement('a');
    link.download = `${stockCode}_analysis.png`;
    link.href = canvas.toDataURL('image/png');
    link.click();
    
    toast.success('导出成功');
  } catch (error) {
    console.error('导出失败:', error);
    toast.error('导出失败，请重试');
  }
};
```

**描述**: 输入无效股票代码(如999999)后，页面显示空数据，没有友好提示  
**期望行为**: 显示"股票代码不存在"或"请输入有效的6位A股代码"

**修复建议**:
```typescript
// 添加代码验证
const validateStockCode = (code: string): boolean => {
  if (!/^\d{6}$/.test(code)) return false;
  
  const prefix = code.substring(0, 3);
  const validPrefixes = ['600', '601', '603', '605', '688', '000', '001', '002', '003', '300', '301'];
  return validPrefixes.includes(prefix);
};

// 搜索时验证
const handleSearch = () => {
```
