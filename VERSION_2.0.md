# Sonata V2.0 - 版本发布说明

**发布日期**: 2026-03-14  
**版本号**: 2.0.0  
**代号**: 全面修复版

---

## 🎯 版本概述

Sonata V2.0 是一个全面修复和优化的版本，解决了 V1.x 中的所有关键问题，包括核心算法、数据一致性、导出功能等。

---

## ✅ 修复清单

### 1. 核心算法修复 (6项)

| 修复项 | 问题 | 解决方案 |
|--------|------|----------|
| RSI 计算 | 使用简单移动平均 | 改为 Wilder's smoothing |
| 布林带 | 使用总体标准差 | 改为样本标准差 (n-1) |
| 夏普比率 | 未考虑无风险利率 | 添加 3% 国债收益率 |
| 盈亏比 | 未考虑交易成本 | 添加 0.126% A股成本 |
| 动量因子 | 不同周期不可比 | 年化标准化处理 |
| 支撑阻力 | 计算方法单一 | 新增6种计算方法 |

### 2. 功能修复 (3项)

| 功能 | 修复内容 |
|------|----------|
| 导出长图 | 添加调试日志、样式处理、进度反馈 |
| 导出 PDF | A4分页、自动处理、高质量压缩 |
| 股票代码验证 | 前缀验证、友好提示、错误处理 |

### 3. 数据一致性修复

| 问题 | 修复方案 |
|------|----------|
| K线图与右侧数据不一致 | 统一使用传入参数 |
| 数字格式化不一致 | 统一使用 toFixed(2) |
| 鼠标悬停数字溢出 | 修复 ECharts formatter |
