### 1. 财务数据增强 ⭐⭐⭐

#### API 升级
- **东方财富 API 字段扩展**
  - 原有字段: 基础行情（价格、成交量）
  - 新增字段:
    - `f9` - 市盈率 (PE)
    - `f162` - 市盈率TTM (PE TTM)
    - `f163` - 市销率 (PS)
    - `f152` - 市净率 (PB)
    - `f164` - ROE (净资产收益率)
    - `f169` - 净利润增长率
    - `f170` - 营收增长率
    - `f116` - 总市值

#### 类型系统升级
- **StockQuote 类型扩展**
  ```typescript
  pe: number;           // 市盈率
  peTtm?: number;       // 市盈率TTM
  pb: number;           // 市净率
  ps?: number;          // 市销率
  peg: number;          // PEG
  roe: number;          // ROE
  profitGrowth: number; // 净利润增长率
  revenueGrowth: number;// 营收增长率
  ```

- **StockData 类型扩展** (StockContext.tsx)
  - 添加所有财务指标字段
  - 支持可选字段，兼容旧数据

#### 新增组件: FinancialMetrics
**功能**: 专业财务指标可视化展示
**特性**:
- 9项核心财务指标显示
- 智能颜色编码（低估/高估识别）
- 估值评价自动计算
- 响应式网格布局

**显示指标**:
1. 市盈率(PE) - 带估值评价
2. 市盈率TTM
3. 市净率(PB)
4. 市销率(PS)
5. PEG
6. ROE - 带等级颜色
7. 净利润增长 - 带趋势颜色
8. 营收增长 - 带趋势颜色
9. 总市值 - 智能格式化

**颜色编码规则**:
- PE < 15: 绿色（低估）
- PE 15-30: 蓝色（合理）
- PE 30-50: 橙色（偏高）
- PE > 50: 红色（高估）

- ROE > 15%: 绿色（优秀）
- ROE 10-15%: 蓝色（良好）
- ROE 5-10%: 橙色（一般）
- ROE < 5%: 红色（较差）

- 增长率 > 30%: 绿色（高增长）
- 增长率 10-30%: 蓝色（稳健）
- 增长率 0-10%: 橙色（低速）
- 增长率 < 0%: 红色（负增长）
