## 📞 问题排查

### 问题 1: 看不到真实数据
**检查**:
1. 浏览器 Console 是否有错误
2. 网络请求是否被 CORS 策略阻止
3. API 是否返回有效数据

**解决**:
- 检查网络连接
- 查看 Console 中的 API 响应
- 确认降级机制是否触发

### 问题 2: 数据源标识不显示
**检查**:
1. `stockData.dataSource` 是否有值
2. UI 组件是否读取了该字段

**解决**:
- 在 StockHeader 或其他组件中添加 `dataSource` 显示
- 检查 React 组件是否正确订阅了 context

### 问题 3: TypeScript 编译错误
**当前状态**:
- StockContext.tsx ✅ 已修复
- 其他文件有未使用变量警告（与本次修复无关）

**解决**:
```bash
# 忽略警告继续构建
npm run build -- --noEmit

# 或者修复其他文件的未使用变量
```
