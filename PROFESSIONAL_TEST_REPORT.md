3. **价格标签**: 每条线都有价格标签

### ⚠️ 问题

**问题 1: 支撑/阻力位计算过于简单**
```typescript
// 当前逻辑：固定百分比
support = currentPrice * 0.95
resistance = currentPrice * 1.08

// 专业做法：基于历史高低点/均线/斐波那契
support = min(low_prices[-20:])  // 20 日最低点
resistance = max(high_prices[-20:])  // 20 日最高点
```

**问题 2: 止损位固定比例不适用于所有股票**
- 高波动股票 (如科技股): 8% 止损可能过小
- 低波动股票 (如银行股): 8% 止损可能过大
- 建议：基于 ATR (平均真实波幅) 动态计算

### 改进建议

**优先级 P1 - 重要改进**:
1. 支撑/阻力位改用真实历史数据:
   ```typescript
   const recentLows = kLineData.slice(-20).map(k => k.low);
   const recentHighs = kLineData.slice(-20).map(k => k.high);
   const support = Math.min(...recentLows);
   const resistance = Math.max(...recentHighs);
   ```

2. 止损位基于 ATR 动态计算:
   ```typescript
   const atr = calculateATR(kLineData, 14); // 14 日 ATR
   const stopLoss = currentPrice - (2 * atr); // 2 倍 ATR 止损
   ```

3. 增加斐波那契回调位标注:
   - 38.2%、50%、61.8% 关键位置
   - 扩展位 127.2%、161.8%
