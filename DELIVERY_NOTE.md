## 当前限制

### 网络限制
- 东方财富板块API有严格反爬机制
- AkShare等Python库无法连接（网络限制）
- 直接HTTP请求被拦截

### 解决方案（需用户配置）
1. **配置代理/VPN**
   - 使用海外IP访问东方财富API
   - 或使用付费代理服务

2. **使用付费数据服务**
   - Wind、Choice等专业金融数据服务
   - 需要购买API授权

3. **手动更新数据**
   - 定期从东方财富网站导出板块数据
   - 保存到 `public/sector_data.json`

## 部署建议

### 生产环境配置
1. **配置代理服务器**
   ```bash
   # 在 vite.config.ts 中配置代理
   proxy: {
     '/api/eastmoney': {
       target: 'https://push2.eastmoney.com',
       changeOrigin: true,
       // 配置代理服务器
     }
   }
   ```

2. **使用付费数据服务**
   - 注册Wind、Choice等数据服务
   - 替换现有API调用

3. **定期手动更新**
   - 每周从东方财富导出板块数据
   - 保存到 `public/sector_data.json`
