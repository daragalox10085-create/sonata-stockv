## 问题诊断

### 当前 StockContext.tsx 的问题

1. **使用随机生成的价格数据**
   - 基础价格硬编码为 3.63 (实际应为 4.082)
   - K 线数据完全随机生成，无真实市场数据

2. **硬编码的基础价格映射**
   ```typescript
   const basePriceMap: Record<string, number> = {
     '513310': 3.63,  // ❌ 错误：实际价格为 4.082
     // ...
   };
   ```

3. **缺失真实 API 集成**
   - 没有调用任何实时数据源
   - 所有数据均为模拟生成

## 修复方案

### 推荐数据源优先级

1. **首选**: 腾讯财经 API (`qt.gtimg.cn`)
   - 优点：简单、快速、数据完整
   - 缺点：HTTP 协议，可能有访问限制

2. **备选**: 东方财富 API (`push2.eastmoney.com`)
   - 优点：HTTPS 协议、JSON 格式、数据准确
   - 缺点：K 线数据可能不完整

### 实现策略

1. **实时价格获取**: 使用腾讯财经或东方财富 API
2. **K 线数据**: 
   - 方案 A: 使用东方财富 K 线 API (需调试参数)
   - 方案 B: 使用历史数据 + 实时价格混合
   - 方案 C: 在无法获取真实 K 线时，基于真实价格生成合理的模拟数据

3. **错误处理**: 
   - API 失败时降级到缓存数据
   - 显示数据更新时间戳
   - 提供手动刷新功能
