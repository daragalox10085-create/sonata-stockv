## 故障排查

### 问题 1: 价格仍然显示 3.63
**原因**: 浏览器缓存了旧代码  
**解决**: 
```bash
# 清除缓存
Ctrl+Shift+Delete (Windows)
Cmd+Shift+Delete (Mac)

# 或强制刷新
Ctrl+F5 (Windows)
Cmd+Shift+R (Mac)
```

### 问题 2: 显示"CORS error"
**原因**: 浏览器安全策略  
**解决**:
1. 开发环境：添加 Chrome 启动参数 `--disable-web-security` (仅测试)
2. 或使用后端代理 (推荐)
3. 或暂时使用备用数据

### 问题 3: API 请求失败
**原因**: 网络问题或 API 限制  
**解决**:
1. 检查网络连接
2. 等待 30 秒后重试
3. 检查 API 是否可用 (使用 curl 测试)

### 问题 4: K 线图不显示
**原因**: 数据格式错误或图表库问题  
**解决**:
1. 检查控制台错误
2. 验证 `kLineData` 数组是否有数据
3. 检查图表组件配置
